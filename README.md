# ChatGPT Outline Navigator

一个用于 ChatGPT 长对话的大纲导航 Chrome 扩展。它会在 ChatGPT 页面右侧生成轻量级刻度栏和对话大纲，帮助你在长线程里快速定位关键提问、跳转到对应消息，并减少反复滚动查找的成本。

## 功能特点

- 右侧刻度导航：将当前对话抽象成可点击的刻度点，适合快速浏览长对话结构。
- Hover 展开大纲：鼠标移入时展开消息摘要列表，移出后自动收起。
- 固定展开模式：可以把大纲面板固定在页面右侧，方便持续导航。
- 当前分支识别：优先解析 ChatGPT 当前 conversation branch，避免把历史分叉消息混入大纲。
- DOM 兜底扫描：当接口数据不可用时，自动从页面可见消息中生成导航项。
- 活跃消息高亮：滚动页面时自动追踪当前位置，并同步高亮最近的大纲项。
- 可配置选项：支持设置大纲条数、面板宽度、右侧偏移、刻度高度、小屏隐藏、默认固定等。
- 低侵入样式：悬浮在页面右侧，不修改 ChatGPT 原始内容。

## 安装使用

### 从 release 包安装

1. 下载 `release/ChatGPT-Outline-Navigator-0.3.1.zip` 并解压。
2. 打开 Chrome 或 Edge 的扩展管理页面。
3. 开启“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择解压后的项目目录。
5. 打开 `https://chatgpt.com/` 或 `https://chat.openai.com/`，进入任意对话页面即可看到右侧导航。

### 从源码安装

1. 克隆仓库：

   ```bash
   git clone https://github.com/acai1998/ChatGPT-Outline-Navigator.git
   cd ChatGPT-Outline-Navigator
   ```

2. 在浏览器扩展管理页面开启“开发者模式”。
3. 选择“加载已解压的扩展程序”，加载项目根目录。

## 使用方式

- 鼠标移动到页面右侧刻度栏，大纲面板会自动展开。
- 点击刻度点或大纲条目，可以跳转到对应消息位置。
- 点击面板右上角的固定按钮，可以保持面板展开。
- 在扩展详情页打开“扩展程序选项”，可以调整显示条数、面板宽度和默认行为。

## 配置项

| 配置 | 说明 |
| --- | --- |
| 大纲条数 | 控制最多展示多少个导航项，默认 10，范围 5-20 |
| 仅显示用户消息 | 开启后只将用户提问作为大纲锚点 |
| 右侧偏移 | 控制导航组件距离页面右侧的像素距离 |
| 面板宽度 | 控制 hover 或固定展开后的大纲面板宽度 |
| 刻度地图高度 | 控制右侧刻度区域的整体高度 |
| 小屏自动隐藏 | 窗口较窄时自动隐藏导航，避免遮挡内容 |
| 默认固定展开 | 打开 ChatGPT 页面后默认展开大纲面板 |
| 显示刻度轨道线 | 控制是否显示右侧导航轨道线 |

## 项目结构

```text

```

## 开发与测试

这个项目是原生 Chrome 扩展，没有构建步骤。修改源码后，在扩展管理页面点击刷新即可重新加载。

运行测试：

```bash
node --test tests/outline-data.test.js
```

测试目前覆盖了 ChatGPT conversation mapping 的当前分支解析逻辑，确保大纲不会错误展示旧分支中的消息。

## 权限说明

扩展仅声明以下权限：

- `storage`：保存用户在选项页中的展示配置。
- `https://chatgpt.com/*` 和 `https://chat.openai.com/*`：在 ChatGPT 页面注入导航组件并读取当前页面对话信息。

项目不会把对话内容上传到第三方服务。对话解析只发生在浏览器本地，并优先使用 ChatGPT 页面同源接口和页面 DOM。

## 适用场景

- 长对话中快速回到某个历史问题。
- 写作、编程、研究类线程中浏览对话脉络。
- 在多轮调试或方案讨论中快速跳转到关键上下文。
